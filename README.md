Computer Graphics Project

A Python application built with Tkinter that implements classic computer graphics algorithms for drawing lines and circles on a 2D canvas.

Features

* DDA Line Algorithm
* Bresenham Line Algorithm
* Midpoint Circle Algorithm

Highlights

* Centered coordinate system (Cartesian-style)
* Custom color selection
* Real-time drawing on canvas
* Display of generated coordinates

Technologies

* Python
* Tkinter

How to Run

Make sure you have Python installed, then run:

```bash
python proyecto_graficacion_centrado.py
```

Description

This project demonstrates fundamental computer graphics techniques by manually implementing drawing algorithms without relying on high-level graphics libraries.

It is designed for educational purposes, helping to understand how pixels are calculated and rendered on screen.
